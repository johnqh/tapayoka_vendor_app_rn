# tapayoka_vendor_app_rn

Vendor native app for iOS/Android. Device setup via BLE, device/order management.

## Stack: Expo + React Native + TypeScript

## Commands

```bash
npm install
npm start             # Metro bundler (port 8085)
npm run android
npm run ios
npm run typecheck
npm run lint
npm test
```

## Key Features

- BLE device setup (react-native-ble-plx)
- Device management
- Order monitoring
